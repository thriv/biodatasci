Data

  1. Option 1: Download all the data. Download and extract this zip file (11.55 Mb) with all the data for the entire workshop. This may include additional datasets that we won’t use here.
  2. Option 2: Download individual datasets as needed.
    • Create a new folder somewhere on your computer that’s easy to get to (e.g., your Desktop). Name it workshop. Inside that folder, make a folder called data, all lowercase.
    • Download individual data files as needed, saving them to the new workshop/data folder you just made. Click to download. If data displays in your browser, right-click and select Save link as… (or similar) to save to the desired location.
